使用 Github Pages + Hexo 搭建自己的博客

Hexo
  • Hexo

Github Pages 可以被认为是用户编写的、托管在 github 上的静态网页。使用Github Pages 可以为你提供一个免费的服务器，免去了自己搭建服务器和写数据库的麻烦。此外还可以绑定自己的域名。

一.安装 Node.js

点击此处访问官网，按需下载相应版本，默认安装可以了

安装好后需要配置环境变量

  • 配置环境变量：（我的node.js的安装路径是 D:\nodejs，各位修改环境变量时，按照自己的安装目录进行修改）
    • 新建变量
      NODE_PATH
      D:\Program Files\nodejs;
    • 修改PATH
      加上%NODE_PATH%

配置成功后，运行 node -v、npm -v 都成功了。
